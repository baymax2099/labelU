# 部署说明

LabelU 支持部署在本地和云端

本地部署见 [安装](/guide/install)。

## Dockerfile 示例

```Dockerfile
FROM python:3.11-slim

WORKDIR /labelu

RUN pip3 install labelu

ENV MEDIA_HOST http://localhost:8000

EXPOSE 8000

CMD ["sh", "-c", "labelu --host=0.0.0.0 --media-host=$MEDIA_HOST"]
```

### 构建镜像

```bash
docker build -t labelu:latest .
```

### 运行容器

```bash
docker run -d -p 8000:8000 --name labelu labelu:latest
```
