# 图片标注组件

我们提供原子化的图片标注组件，可以帮助你快速构建图片标注功能。


## 基本

最基础的图片标注组件，提供了图片的基础标注功能，提供更高的自定义能力。

```bash
npm i @labelu/image
```

## 带 UI 框架

此组件提供了一个包含UI框架的图片标注组件，非常适合快速构建图片标注功能。也可以很方便地嵌入到已有的 React 项目中。

