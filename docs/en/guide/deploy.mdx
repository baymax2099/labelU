# Deployment

LabelU supports deployment both locally and in the cloud.

For local deployment, see [Installation](/guide/install).

## Dockerfile Example

```Dockerfile
FROM python:3.11-slim

WORKDIR /labelu

RUN pip3 install labelu==1.1.4

ENV MEDIA_HOST http://localhost:8000

EXPOSE 8000

CMD ["sh", "-c", "labelu --host=0.0.0.0 --media-host=$MEDIA_HOST"]
```

### Build Image

```bash
docker build -t labelu:1.1.4 .
```

### Run Container

```bash
docker run -d -p 8000:8000 --name labelu labelu:1.1.4
```
