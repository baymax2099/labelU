# 多人协作标注

:::info

协同标注需要 1.2.0 以上的版本才支持。如果需要多人使用，建议将数据库迁移为 MySQL，具体请查看 [迁移数据库](/guide/migrate)。

:::

## 如何配置？

在任务的样本列表页面，点击“协同标注”出现如下界面：

![协同标注](/zh/collaboration-1.png)

勾选需要加入标注的用户即可生效（将任务链接复制后，发送给参与标注的用户，局域网内将 `localhost` 替换为本机 ip 地址，用户注册后刷新用户列表即可）。参与标注的用户即可在任务列表中看到该任务，如下图所示：

![协同标注](/zh/collaboration-2.png)

:::tip

非任务创建者无法修改任务配置和删除任务。

:::

当某个样本被其他用户标注时，进入该样本标注页面后将无法标注，如下图所示：

![协同标注](/zh/collaboration-3.png)

该用户离开该样本标注页面后，其他用户即可继续标注，同时会刷新该样本的标注数据，如下图所示：

![协同标注](/zh/collaboration-4.png)

最终，任务的标注者将会显示在样本列表页：

![协同标注](/zh/collaboration-5.png)
